# Isometry Lab Website

Product website for [Isometry Lab](https://github.com/shanyang-me/isometry) — physics world models that understand isometry laws.

## Local Development

```bash
cd ~/isometry-website
python -m http.server 8000
# Open http://localhost:8000
```

## Deploy

Hosted on GitHub Pages. Push to `main` and enable Pages in repo settings (source: root of `main` branch).

## Structure

```
index.html          # Single-page site
css/style.css       # Dark theme, responsive
js/main.js          # Nav, smooth scroll, demo interaction
assets/favicon.svg  # Favicon
```
